als2sndj — Ableton / MIDI / MML ⇆ .sndj

Drop an Ableton Live Set (.als), a Standard MIDI File (.mid), or MML text (.mml/.txt) to get a .sndj song (load it with the savetool or the FILES screen). Drop a .sndj to go the other way: a minimal Session-view .als, or MML text below. Everything runs locally.
