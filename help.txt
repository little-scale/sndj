NAVIGATION:
DPAD         MOVE THE CURSOR
HOLD A +DPAD GO TO SCREENS
A + B        PLAY THIS SCREEN
START        PLAY / STOP ALL
SELECT       LIVE MODE
HOLD A 3 SEC ENTER/EXIT HELP

CELL EDITING:
TAP B        INSERT ON EMPTY
HOLD B + L/R EDIT, SMALL STEP
HOLD B + U/D EDIT, BIG STEP
HOLD B TAP A CUT WITH COPY
> TAP B,B    PASTE VALUE

---
BLOCK EDITING:
HOLD Y TAP B SELECT BLOCK
> DPAD       EXTEND BLOCK
> TAP B      COPY BLOCK
> B + A      CUT BLOCK
> TAP B,B    PASTE ELSEWHERE

MINT AND CLONE:
TAP B,B ON EMPTY REFERENCE
= MINT NEW PHRASE/CHAIN

TAP B,B ON EXISTING
= CLONE PHRASE/CHAIN

CHANNELS:
L / R        PREV/NEXT TRACK
X + DPAD     MUTE / SOLO

---
RELATIONSHIPS:
TRACKS DO NOT OWN INSTRUMENTS
TRACKS DO NOT OWN NOTE DATA
INSTRUMENTS, PHRASES, CHAINS
AND TABLES ARE GLOBAL POOLS

STRUCTURE:
NOTES LIVE IN PHRASES
PHRASES LIVE IN CHAINS
CHAINS SIT ON TRACKS

THE GROOVE IS ONE 2-STEP PAIR
6/6 STRAIGHT - 8/4 SWING

AUTOMATION:
TABLES RUN COMMANDS PER TICK
VIB/TRM ARE INSTRUMENT LFOS

---
INSTRUMENT TYPES:
SMP  A POOL SAMPLE, PITCHED
     ACROSS THE KEYBOARD -
     LOOP OVERRIDES THE POOL

KIT  THE NOTE PICKS A DRUM
     SLOT (BUILD ON KIT)

WAV  A DRAWN 32-STEP WAVE -
     8 BANKS, B SWITCHES

NSE  NOISE - THE NOTE SETS
     THE SHARED NOISE CLOCK

SLC  ONE SAMPLE CUT INTO N
     SLICES, THE NOTE PICKS

KARP THE ECHO LOOP PLUCKED AS
     A STRING - ONE PER SONG
ANY TYPE ON ANY OF 8 VOICES

---
SAMPLES AND MEMORY:
THE CART ROM HOLDS THE POOL -
UP TO 48 SAMPLES, REPLACED
WITH THE BROWSER PATCHER.

AUDIO RAM IS 64K SHARED BY
SAMPLES, ECHO AND THE DRIVER.
A SAMPLE ONLY SOUNDS ONCE IT
IS LOADED INTO AUDIO RAM.

LOADED AT BOOT = THE 8 BOOT
INSTRUMENTS + EVERY KIT SLOT.

EDIT ANY INSTRUMENT OR KIT TO
POINT AT A POOL SAMPLE - IT
LOADS ON THE SPOT. LOADING A
SONG LOADS EXACTLY ITS SET.

ECHO SHARES THE RAM - 2K PER
DELAY STEP (SEE ECHO SCREEN).
